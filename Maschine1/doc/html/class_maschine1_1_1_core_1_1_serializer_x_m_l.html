override void Maschine1.Core.SerializerXML.WriteElementStart ( string  Name)
virtual

Starts a section. A section represents a piece of complex data (f.e. an object). After starting a section you can WriteData to push the attributes of the object on the stream. Its possible to nest sections. You have to use WriteElementEnd to indicate that you have finished on the object.
